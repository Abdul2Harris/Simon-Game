## üïπÔ∏è How to Play
1. **Start the Game:** Press any key on your keyboard to begin.
2. **Watch the Sequence:** The game will flash a button and play a sound.
3. **Repeat the Pattern:** Click the buttons in the exact same order as shown.
4. **Level Up:** Every time you successfully complete a pattern, the sequence gets longer and faster!
5. **Game Over:** If you click the wrong button, the game ends. Press any key to restart and try to beat your high score.
